# Responsive Accessible Webpage

A responsive and accessible website created for the SBA2 project. This project demonstrates the use of semantic HTML, CSS, responsive design techniques, and accessibility best practices to create a modern, user-friendly webpage.

## Features

- Semantic HTML5 structure
- Responsive layout for desktop, tablet, and mobile devices
- Accessible navigation
- Keyboard-friendly navigation
- Responsive images
- CSS Flexbox and Grid layouts
